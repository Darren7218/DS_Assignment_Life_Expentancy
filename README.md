**UCCC3073 Data Science Assignment** A comprehensive Data Science project utilizing the **Life Expectancy (WHO)** dataset from Kaggle. This project explores the demographic, economic, and social factors that influence life expectancy across different countries, applying data analysis and machine learning techniques via Jupyter Notebook.

---

## ✨ Project Overview

This assignment focuses on the end-to-end data science lifecycle, analyzing global health data to extract meaningful insights. The primary objective is to understand how various immunization factors, mortality rates, economic indices, and social factors impact the life expectancy of a population.

### Key Workflows:
* **🧹 Data Preprocessing & Cleaning:** Handling missing values, dealing with outliers, and normalizing data to prepare the WHO dataset for accurate modeling.
* **📊 Exploratory Data Analysis (EDA):** Visualizing correlations between life expectancy and features like GDP, schooling, adult mortality, and alcohol consumption.
* **⚙️ Feature Engineering:** Selecting the most impactful variables to reduce noise and improve model performance.
* **📈 Predictive Modeling:** Implementing regression techniques to predict life expectancy based on historical country data.
